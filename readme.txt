# 🥋 El Bruto - Recreación en PHP y JavaScript

Una recreación no oficial del clásico juego **El Bruto** usando tecnologías web como **PHP**, **JavaScript**, **HTML** y **CSS**. Crea tu bruto, enfréntate a otros jugadores, sube de nivel y desbloquea nuevas habilidades, armas y mascotas.

## 🚀 Características

- Creación de personajes con atributos aleatorios
- Sistema de combate automático estilo turn-based
- Subida de nivel y progresión de personajes
- Habilidades desbloqueables y equipamiento
- Mascotas que acompañan en combate
- Sistema de ranking y clasificación
- Interfaz responsiva y animaciones con JS

## 🛠️ Tecnologías utilizadas

- **PHP** – Backend y lógica del juego
- **JavaScript** – Interacciones del cliente y animaciones
- **SQL** – Base de datos
- **HTML/CSS** – Interfaz del usuario
